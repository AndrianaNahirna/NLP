# Лабораторна робота 9: Word Embeddings (Word2Vec / FastText)

**1. Який корпус використано**
Аналізувався очищений та лематизований корпус клієнтських відгуків про українські інтернет-магазини електроніки (Rozetka, Citrus). Розмір корпусу — 3034 документи (близько 600 000 токенів).

**2. Які моделі натреновано**
Було натреновано дві моделі векторних представлень слів:
* Word2Vec (архітектура Skip-gram)
* FastText (архітектура Skip-gram)

**3. Які параметри використано**
Для чесного порівняння обидві моделі тренувалися з однаковими базовими параметрами:
* `vector_size = 100`
* `window = 5`
* `min_count = 3`
* `sg = 1` (Skip-gram обрано для кращої роботи з рідкісними словами)

**4. Які типи слів аналізували**
Для аналізу найближчих сусідів (Nearest Neighbors) використовувалися: часті загальні слова, доменні терміни, слова з морфологічною варіативністю, сленг/скорочення та слова із шумом (опечатки, трансліт).

**5. Які 5 кейсів виявились найцікавішими**
1. **"розетко" (Шум):** Обидві моделі впоралися ідеально (знайшли трансліт бренду).
2. **"цитрус" (Конкуренти):** Word2Vec блискуче знайшов прямих конкурентів магазину (Moyo, Stylus).
3. **"доставка" (Логіка vs n-грами):** Word2Vec видав логістику ("кур'єр", "самовивіз"), а FastText схибив, видавши слова на "дост-" ("достатньо", "доступ").
4. **"навушник" (Морфологічний збій):** FastText помилково згрупував гаджет зі словом "рушник" через однакове закінчення.
5. **"глючити" (Сленг):** Word2Vec знайшов синоніми несправностей ("вимикатися"), а FastText видав антоніми ("включити", "відключити") через суфікс.

**6. Де FastText був кращий / де ні**
* **Кращий:** У нормалізації брудного тексту. Він чудово знаходить опечатки (наприклад, усі неправильні написання слова "менеджер") та не боїться невідомих слів (OOV) завдяки n-грамам символів.
* **Гірший:** У розумінні семантики. Він часто ігнорує сенс слова, групуючи його з іншими лише на основі візуальної схожості літер (як у випадках з "навушник" та "доставка").

**7. Чи embeddings реально корисні для вашого корпусу**
Так, вони виявилися дуже корисними. Ембеддинги (особливо Word2Vec) дозволяють глибоко аналізувати клієнтські болі (наприклад, пов'язуючи "коробку" зі словами "брудна" та "відкрита"), а також допомагають автоматично розширювати словники тональності синонімами та технічним сленгом.
