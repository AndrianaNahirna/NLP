# Лабораторна робота 6: TF-IDF + Logistic baseline

**1. Напрям проєкту:**
Напрям А (Класифікація текстів).

**2. Classification-підзадача:**
Бінарна класифікація тональності клієнтських відгуків (positive vs negative) на основі відкритих даних з українських маркетплейсів.

**3. Порівняння baseline-варіантів:**
В рамках експерименту було побудовано та порівняно 3 варіанти:
- **Baseline 1:** Базовий очищений текст (`processed_v2`) + стандартна Logistic Regression.
- **Baseline 2:** Лематизований текст (`lemma_text`) + стандартна Logistic Regression.
- **Baseline 3 (Переможець):** Лематизований текст (`lemma_text`) + Logistic Regression з балансуванням класів (`class_weight='balanced'`).

**4. Основні цифри (метрики на test set):**
- **Baseline 1:** Accuracy = 0.8092, Macro-F1 = 0.7597 
- **Baseline 2:** Accuracy = 0.8059, Macro-F1 = 0.7582 
- **Baseline 3 (Найкращий):** Accuracy = 0.8553, Macro-F1 = 0.8407 
*(Використання балансування класів дало значний приріст Macro-F1 завдяки кращому розпізнаванню негативного класу).*

**5. Результати Error Analysis та наступні кроки:**
Ручний розбір помилок найкращої моделі показав наступне:
- **Шумна розмітка (Noisy labels):** Часто зустрічаються абсолютно гнівні відгуки, яким користувач випадково або помилково поставив позитивну оцінку. Модель класифікує текст правильно, але метрика падає через хибний "золотий стандарт".
- **Overlap класів:** Модель на базі TF-IDF погано справляється зі зміною тональності всередині тексту (наприклад, коли відгук починається з похвали, але після слова "але" йде скарга).
- **Leakage (Витік даних):** У топ позитивних фіч потрапило слово "цитрус", що свідчить про запам'ятовування моделлю кореляції конкретного бренду з тональністю.

**Що будемо фіксити далі:**
1. Очистити датасет від явно хибної розмітки.
2. Видалити або замаскувати назви конкретних магазинів та брендів (додати у стоп-слова або використати NER), щоб усунути leakage.
3. Для вирішення проблеми "overlap класів" доцільно перейти від TF-IDF до контекстних ембеддингів або трансформерів (BERT).
